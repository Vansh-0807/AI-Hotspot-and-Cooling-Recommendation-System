Project Workflow

Development Timeline

Phase 1
Data Collection
Acquired 12,847 LISS-IV scenes from Bhuvan portal spanning 2020–2024 across varied cloud conditions.
Phase 2
Preprocessing
Radiometric correction, co-registration, and cloud masking using Fmask 4.0 with manual validation.
Phase 3
Model Training
Fine-tuned Diffusion + GAN + ViT ensemble on 4× NVIDIA A100 GPUs for 340 epochs over 12 days.
Phase 4
Validation
Blind test on 1,200 held-out scenes. SSIM 0.982, PSNR 41.7 dB, spectral RMSE 0.021 achieved.
Phase 5
Deployment
Production API serving 1.8s inference per tile, integrated with Bhuvan geoportal pipeline.
